Ruby's Enumerable.map method

May 18, 2014

Array.map{|n| perform actions here}

The map method is a Ruby method that applies to Enumerables, which includes Arrays, Hashes, and Ranges. The map method takes each of the values in Enumerable, performs a set of procedures which are defined in a block, and returns the new values in an Array.

[1,2,3,4].map{|n| 2*n} # => Returns [2,4,6,8]

In the above example, the procedures defined within the block multiplies each of the values in the array by 2. The modified values are then put into a new array and returned.

The map method is not only used by arrays of numbers; it can also be used on arrays of strings, as in the following example:

["apple", "spaghetti"].map{|n| n.length} # => Returns [5, 9]

The map method is similar to the inject/reduce method in that it takes a range, hash, or array and performs actions based on the content of a block. However, while the map method returns an array, the inject/reduce method returns a single value. As the name of the method suggests, inject/reduce is used to reduce a multi-value array to a single value.
