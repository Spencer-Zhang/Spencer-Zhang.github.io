Inline, Block, and Inline-Block

May 2, 2014

Inline elements can be used to style text within a paragraph. The shape of an inline element is determined by the content of the element, so their size cannot be set via the height and weight parameters. In addition, they are only affected by the left and right margin and padding properties.

Block elements give you more control over the shape of an element, allowing you to set its width and height, and including full functionality of the margin and padding properties. However, they automatically include a line break at the end of each element. If we were to have several block elements on a page, they will be positioned on top of each other by default.

Often, we will want to have elements with a definable width and height like a block element, but that can also be place side by side. In order to do this, we need to use an Inline-Block element, which we can create by using "display:inline-block". Inline-Block elements are almost the same as Block elements, except for the fact that they don't include the line break at the end.
